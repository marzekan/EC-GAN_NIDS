# Install

Clone the repo.
```bash
git clone https://github.com/marzekan/EC-GAN_NIDS.git
```

Install dependencies.
```bash
cd EC-GAN_NIDS/
```
```bash
pipenv install
pipenv run jupyter notebook
```
or

```bash
pip install -r requirements.txt
jupyter notebook
```
